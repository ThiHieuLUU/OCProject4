14 """Python Abstract Syntax Tree New Generation
15  
16 The aim of this module is to provide a common base representation of
17 python source code for projects such as pychecker, pyreverse,
18 pylint... Well, actually the development of this library is essentially
19 governed by pylint's needs.
20  
21 It extends class defined in the python's _ast module with some
22 additional methods and attributes. Instance attributes are added by a
23 builder object, which can either generate extended ast (let's call
24 them astroid ;) by visiting an existent ast tree or by inspecting living
25 object. Methods are added by monkey patching ast classes.
26  
27 Main modules are:
28  
29 * nodes and scoped_nodes for more information about methods and
30 attributes added to different node classes
31  
32 * the manager contains a high level object to get astroid trees from
33 source files and living objects. It maintains a cache of previously
34 constructed tree for quick access
35  
36 * builder contains the class responsible to build astroid trees
37 """

1 """
2 Key bindings registry.
3  
4 A `Registry` object is a container that holds a list of key bindings. It has a
5 very efficient internal data structure for checking which key bindings apply
6 for a pressed key.
7  
8 Typical usage::
9  
10 r = Registry()
11  
12 @r.add_binding(Keys.ControlX, Keys.ControlC, filter=INSERT)
13 def handler(event):
14 # Handle ControlX-ControlC key sequence.
15 pass
16  
17  
18 It is also possible to combine multiple registries. We do this in the default
19 key bindings. There are some registries that contain Emacs bindings, while
20 others contain the Vi bindings. They are merged together using a
21 `MergedRegistry`.
22  
23 We also have a `ConditionalRegistry` object that can enable/disable a group of
24 key bindings at once.

32 """Checker for Sphinx, Google, or Numpy style docstrings
33  
34 * Check that all function, method and constructor parameters are mentioned
35 in the params and types part of the docstring. Constructor parameters
  • E501 Line too long (80 > 79 characters)
36 can be documented in either the class docstring or ``__init__`` docstring,
37 but not both.
38 * Check that there are no naming inconsistencies between the signature and
  • E501 Line too long (80 > 79 characters)
